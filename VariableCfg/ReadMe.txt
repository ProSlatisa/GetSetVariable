-----VariableCfg by Jason Lin-----
Get variable: VariableCfg.exe /g VariableName GUID size(default 4 bytes)
Set variable: VariableCfg.exe /s VariableName GUID
Delete variable: VariableCfg.exe /d VariableName GUID
-----------------------------------
See Example: VariableCfg.exe /e
